### 1.git基本使用手册

- git  clone
- git  init
- git add 目录/文件名
- git commit  -m  "注释"  
- ssh-keygen -t rsa -C "1213144@qq.com"   生成密钥并复制到个人key
-  git remote add 仓库名 +仓库地址

### 2.分支操作

- git push  

- git push origin newbranch   推送数据 

- git branch -d  newbranch    删除分支

- git pull   拉取数据

- git merge  合并分支
